네이버 뉴스 키워드 크롤링 서비스

해당 서비스는 메일 주소와 특정 단어를 입력하시면, 자동으로 매일 네이버 주요 뉴스를 검색하여 해당 단어가 포함된 기사의 제목과 링크, 본문을 취합하여 메일을 송부하는 서비스 입니다.

검색된 기사 수는 첫 페이지 (최대 10개) 이며, 네이버 뉴스 포맷으로 서비스 되는 기사만 취합됩니다.

따라서 대부분의 해외 원문 뉴스는 제외 됩니다.


원하시는 서비스를 선택하십시오



네이버 주식 종목 크롤링

주식 종목 코드 6자리 (ex. 삼성전자 : 005930) 과 받으실 e-mail를 입력해 주시면 관련정보를 실시간으로 크롤링하여 메일 첨부파일로 승부 합니다.
